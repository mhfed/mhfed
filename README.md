### 👨‍💻 Về bản thân

Với 5 năm kinh nghiệm trong mảng e-commerce và fintech, tôi tập trung vào việc thiết kế kiến trúc hệ thống Frontend quy mô lớn, tối ưu hóa hiệu năng (Core Web Vitals) và xây dựng các công cụ hỗ trợ developer. 

- 🚀 **Hiện tại:** Đang đảm nhiệm vai trò Frontend Engineer tại **H&L Consulting**, tập trung phát triển dự án toàn cầu **NFL Flag**.
- 🛠️ **Mã nguồn mở & Công cụ (Creator):** Tác giả của thư viện npm `next-query-sync` và công cụ CLI viết bằng Rust `safe-rm-hehe`.
- ⚡ **Thành tựu nổi bật:** Từng dẫn dắt quá trình chuyển đổi kiến trúc hệ thống từ Laravel sang Next.js (App Router), cải thiện điểm PageSpeed từ 29 lên 90+ và đóng góp trực tiếp vào việc tăng Conversion Rate.
- 🤖 **Workflow:** Ứng dụng tích cực AI-Augmented Development (Cursor, Claude Code, Zed) để tăng tốc độ và chất lượng phát triển phần mềm.
- 🎯 **Định hướng:** Xây dựng kiến trúc high-performance, phát triển Zalo Mini App, và lập trình hệ thống với Rust.
